ChronoCanvas - Full MCA Project (Polished Theme)
How to run:
1. Extract folder.
2. Open folder in VS Code.
3. Install Live Server extension.
4. Right-click index.html -> Open with Live Server.
Default admin: username=admin password=admin
Notes:
- Images/files are stored in IndexedDB (use devtools -> Application -> IndexedDB to inspect)
- Metadata (users/items/pending) in localStorage.
- Use Admin panel to approve pending submissions.
